## Design Specifications

**Visual Theme**: Brushed steel/metal plaque aesthetic
- Conveys durability, institutional integrity, and permanence
- Reflects the foundation-building nature of the framework
- Professional and trustworthy presentation

**Core Symbol**: The Hodge Gate
- Oval design representing the one-way valve
- Where systemic debt enters and wealth is released
- Central to the GSRT operational architecture

**Consistent Messaging**:
All logos carry the unified mission statement:
- **Tagline**: "Human Advocacy • Liberative Operations"
- **Core Message**: "Forgive Yesterday • Heal Today • Thrive Tomorrow Together"

## Usage Guidelines

These logos should be used across:
- Government partnership proposals
- First Nations collaboration materials
- Investor and philanthropist presentations
- Media and documentary coverage
- Official GSRT framework documentation
- Pilot project signage (Don River corridor)
